Launch of My Site

This post marks the launch of my new website. The previous one was a terrible wordpress monstrosity that I left up for way too long.

This website started as / was inspired by Lihaoyi's 15 Minute Blog Engine Some parts of the source code are largely untouched from that guide, but this is overall my own thing.

I didn't write a line of HTML or CSS for this page - its all scala and markdown. I just run my Generator.sc, and all the pages are freshly generated for every change.

ScalaTags is a fun thing to mess around with. It provides a way to functionally compose a website and all its styles.

It's a good tool for side-projects, especially combined with Ammonite, which makes it possible to build your project without relying heavier tools like SBT or Maven.
